A voice agent built with ElevenLabs and Cursor AI

## Quick Start

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Set up environment variables:**
   ```bash
   cp .env.example .env
   # Edit .env with your ElevenLabs API key
   ```

3. **Run the application:**
   ```bash
   npm start
   ```
